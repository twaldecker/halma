# A Halma game

Based on a Feathers Backend and a React frontend.

## Development

```
cd frontend
yarn install
yarn start
```

```
cd backend
yarn install
yarn dev
```

## Docker

```
docker build -t halma .
docker container run -p 3030:3030 halma
```
